# メシマッチ v0.35

店舗名タップで開く全画面詳細ページの初版です。

## 起動方法

`index.html` をブラウザで開くだけでも動きます。

PWAとしてのホーム画面追加・Service Workerによる自動更新は、GitHub PagesなどのHTTPS環境で確認してください。

## ZIP構成

このZIPは、ZIP直下にファイルを直接配置しています。

```text
meshi_match_v35.zip
├─ index.html
├─ style.css
├─ app.js
├─ README.md
├─ manifest.json
├─ service-worker.js
├─ icon-192.png
└─ icon-512.png
```

## v0.35の追加内容

- 店舗名タップで全画面詳細ページを開く
- 右上の×で閉じる
- 詳細ページは以下の入口を統一
  - スワイプ画面
  - Like履歴
  - 決定履歴
  - 友達との候補一覧
- 詳細ページの上部に大きめ写真エリア
- 複数枚写真
  - 右タップで次
  - 左タップで前
  - 下にインジケーター
- 写真がない店でも、仮表示で詳細ページを開ける
- 店名の下に「店舗情報」ブロックを配置
- 店舗情報で以下を表示
  - 営業状態（営業中 / 営業時間外 / 営業時間情報なし）
  - 今日の営業時間
  - 住所 1行表示
  - Mapsボタン
  - テイクアウト可否
  - 簡単なメニュー（最大3つ、ある店だけ）
  - 席数（ある店だけ）
- 詳細ページでは Like / Nope / Superlike を置かない
- 訪問状態は詳細ページから変更可能

## 注意

v0.35では、周辺取得した実店舗の写真・営業時間・テイクアウト情報が不足する場合があります。  
その場合は以下の扱いです。

- 写真なし → 仮表示
- 営業時間なし → 営業時間情報なし
- テイクアウト情報なし → テイクアウト情報なし

## キャッシュ対策

```html
<link rel="stylesheet" href="style.css?v=0.35" />
<script src="app.js?v=0.35"></script>
```

Service Workerのキャッシュ名も `meshi-match-v0.35` に更新しています。

## 確認ポイント

1. スワイプ画面で店名をタップすると全画面詳細が開くか
2. 右上の×で閉じるか
3. Like履歴や決定履歴からも同じ詳細ページを開けるか
4. 写真エリアで左右タップとインジケーターが動くか
5. 営業状態・今日の営業時間・住所・Maps・テイクアウト・メニュー・席数が見えるか
